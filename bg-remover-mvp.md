## 一、项目概述

### 1.1 产品简介

Image Background Remover 是一款在线图片背景去除工具，用户只需上传图片，即可一键自动去除背景，下载透明背景的 PNG 图片。

### 1.2 目标用户

- 电商卖家（商品主图抠图）
- 设计师（快速处理素材）
- 普通用户（证件照、头像处理）
- 社交媒体创作者（内容制作）

### 1.3 核心价值

- **快速**：秒级出结果，无需等待
- **简单**：无需注册，上传即用
- **免费**：基础功能完全免费

## 二、技术架构

### 2.1 整体架构

```
用户上传图片
    ↓
Cloudflare Pages（前端 Next.js）
    ↓
Cloudflare Workers（后端，内存处理）
    ↓
remove.bg API（AI 去背景）
    ↓
返回处理后图片给用户下载
```

### 2.2 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Next.js 14 + TailwindCSS | Cloudflare Pages 部署 |
| 后端 | Cloudflare Workers | 无服务器，内存处理 |
| AI 引擎 | remove.bg API | 专业抠图服务 |
| 存储 | 无 | 图片全程在内存中处理，不落盘 |
| CDN | Cloudflare | 全球加速 |

### 2.3 关键约束

- 图片大小限制：≤ 10MB
- 支持格式：JPG、PNG、WebP
- 不存储任何用户图片（隐私保护）

## 三、功能需求

### 3.1 F1 - 图片上传

- 用户可通过点击或拖拽方式上传图片
- 支持格式：JPG、PNG、WebP
- 文件大小：≤ 10MB
- 拖拽文件到上传区域 / 点击选择文件
- 显示上传进度
- 文件不合规时给出明确错误提示

### 3.2 F2 - 背景去除

- 上传完成后自动调用 remove.bg API 处理图片
- 全程内存处理，不存储文件
- 处理中：显示 Loading 动画
- 处理成功：展示结果图
- 处理失败：显示错误信息并允许重试

### 3.3 F3 - 结果预览

- 并排展示原图与处理后图片
- 处理后图片背景显示棋盘格纹（表示透明）
- 支持 Before/After 对比滑块

### 3.4 F4 - 图片下载

- 用户可下载透明背景的 PNG 图片
- 文件名：`removed-bg.png`
- 点击"下载"按钮即可保存到本地

### 3.5 错误处理规范

| 错误场景 | 提示文案 |
|----------|----------|
| 文件格式不支持 | 请上传 JPG、PNG 或 WebP 格式的图片 |
| 文件过大 | 图片大小不能超过 10MB |
| 网络超时 | 网络异常，请检查后重试 |
| API 额度耗尽 | 服务繁忙，请稍后再试 |
| 未知错误 | 处理失败，请重试 |

## 五、API 设计

### 5.1 背景去除接口

**Endpoint**

```
POST https://your-worker.workers.dev/remove-bg
```

**Request**

```
Content-Type: multipart/form-data
Body: { image: <File> }
```

**Response 成功**

```
Status: 200
Content-Type: image/png
Body: <PNG 二进制数据>
```

**Response 失败**

```json
{
  "error": "错误描述"
}
```

### 5.2 HTTP 状态码

| 状态码 | 含义 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误（未上传图片） |
| 413 | 文件过大 |
| 429 | API 额度不足 |
| 500 | 服务器内部错误 |
